Correlação do tráfego de rede com o uso da CPU

O modelo foi treinado por uma machine learning e se trata de uma regressão linear simples, onde relaciona os dados recebidos pela rede (em GB) com base no percentual de uso da CPU. A linha vermelha no mostra como a quantidade de dados recebidos pela rede pode influenciar o uso da CPU.

A regressão linear demonstra que, à medida que os dados recebidos da rede aumenta, o uso da CPU tende a aumentar também. Com base nessa relação, é essencial ajustar a capacidade da rede conforme o tráfego de dados para evitar sobrecargas dos componentes e melhorar o desempenho do sistema. O coeficiente de determinação (R²) é de 0.933, o que indica que o modelo tem alta precisão e é confiável.
